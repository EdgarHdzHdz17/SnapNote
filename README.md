## Run Locally

Clone the SnapNote

```bash
  git clone https://github.com/EdgarHdzHdz17/SnapNote.git
```

Go to the project directory

```bash
  cd SnapNote
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npx sxpo start
```
